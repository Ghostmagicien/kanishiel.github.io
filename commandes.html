Commandes de KaniShiel

Les commandes relatives aux jeux de KaniShiel sont disponibles dans Règles

Le préfixe de KaniShiel est k- sur tous les serveurs.


⏣ balance - Permet d'afficher sa monnaie

    USAGE
    k-balance / k-balance {@user}

    Permet d'afficher la quantité de Wolfard et d'Arcade Coins que l'on possède.
    Permet également d'afficher notre position dans le classement du serveur.
    Si l'on ajoute une mention après la commande, cela affiche la money de la personne mentionnée.

    EXEMPLE
    k-balance @Magicien_ghost

⏣ pay - Permet de donner de l'argent à quelqu'un

    USAGE
    k-pay {@user} (Wolfard/Arcade) {somme}

    Permet d'envoyer des Wolfards ou des Arcade Coins à n'importe qui présent sur le serveur

    EXEMPLE
    k-pay @Magicien_ghost Wolfard 100

⏣ channelinfos - Permet d'afficher les infos d'un salon

    USAGE
    k-channelinfos [#salon] / k-ci [#salon]

    Permet l'affichage de toutes les informations relatives à un salon.
    Affiche les infos du salon où l'on se trouve si l'on ne nomme pas de salon en particulier

    EXEMPLE
    k-channelinfos #général

⏣ emojiinfos - Permet d'afficher les infos d'un emoji

    USAGE
    k-emojiinfos {emoji} / k-ei {emoji}

    Permet l'affichage de toutes les informations relatives à un emoji.

    EXEMPLE
    k-emojiinfos 👍

⏣ gamelist - Permet d'afficher les parties en cours avec KaniShiel

    USAGE
    k-gamelist (werewolf/monopoly/millebornes)

    Permet l'affichage des parties en cours avec KaniShiel.
    Affiche également un lien d'ivitation vers les serveurs publics.

    EXEMPLE
    k-gamelist werewolf

⏣ help - Permet d'afficher le panel d'aide

    USAGE
    k-help [commande] / k-h [commande]

    Permet l'affichage de toutes les commandes disponibles
    Si l'on ajoute une commande après, cela affiche le fonctionnement de cette commande.

    EXEMPLE
    k-help pay

⏣ invite - Permet d'accéder aux liens du bot

    USAGE
    k-invite / k-support

    Permet l'affichage en message privé du lien d'invitation de KaniShiel.
    Permet également l'affichage du lien d'invitation du serveur de support.

    EXEMPLE
    k-invite

⏣ top - Permet d'afficher les classements disponibles

    USAGE
    k-top (wolfard/arcade/level)/ k-leaderboard (wolfard/arcade/level)

    Permet d'afficher les personnes ayant le plus d'Arcade Coins sur le serveur.
    Ainsi que votre position si vous n'êtes pas dans le top 10.

    EXEMPLE
    k-top arcade

⏣ ping - Permet d'afficher la latence du bot

    USAGE
    k-ping

    Permet d'afficher la vitesse de réponse de KaniShiel en ms.

⏣ premium - Permet d'afficher les infos du Premium

    USAGE
    k-premium / k-prem

    Permet d'afficher toutes les informations relatives au premium.

⏣ rank - Permet d'afficher son niveau et sa position sur le serveur

    USAGE
    k-rank [@user]/ k-level [@user]

    Permet d'afficher son niveau ainsi que sa position sur le serveur actuel
    Accompagnée d'une mention, elle permet de voir le niveau de quelqu'un d'autre.

    EXEMPLE
    k-rank @Magicien_ghost

    USAGE

⏣ suggest - Envoie une suggestion

    USAGE
    k-suggest {suggestion}

    Permet de suggérer une idée pour améliorer Kani ou les jeux qu'il propose.
    /!\ Les messages envoyés font sur le serveur de support, tout troll sera puni /!\

    EXEMPLE
    k-suggest Voici mon idée pour améliorer le bot...

⏣ useredit - Permet de modifier son profil

    USAGE
    k-useredit colortext {color-hex} / Réservé premium k-useredit (title/background)

    Permet de modifier la couleur d'écriture de son profil.
    Réservé premium: permet de modifier notre descritpion ainsi que la photo à l'arrière plan.

    EXEMPLE
    k-useredit colortext #FF0000

⏣ userinfo - Permet de voir son profil

    USAGE
    k-userinfo [@mention] / k-ui [@mention]

    Permet de voir son profil ou celui d'une autre personne.

    EXEMPLE
    k-userinfo @Magicien_ghost

⏣ gameban - Permet de bannir un membre de l'utilisation de KaniShiel

    USAGE
    k-gameban {@mention} (game/all) [t-temps] [r-raison]

    Permet d'interdire l'utilisation de KaniShiel à un joueur.
    Il ne pourra plus intéragir avec Kani.

    EXEMPLE
    k-gameban @Magicien_ghost werewolf t-3w r-ban

⏣ gameunban - Permet de débannir un membre de l'utilisation de KaniShiel

    USAGE
    k-gameunban {@mention} (game/all) [r-raison]

    Permet d'autoriser de nouveau l'accès à un joueur banni.
    Il pourra de nouveau intéragir avec Kani.

    EXEMPLE
    k-gameunban @Magicien_ghost werewolf r-ban

    EXEMPLE
